# House Price Prediction — Linear Regression

This project builds and evaluates a Linear Regression model on the California Housing dataset.

Contents:
- `task1_ml_linear_regression.ipynb` — Jupyter notebook with the full workflow (EDA, training, evaluation, visualization, save model).
- `california_housing.csv` — dataset used (if present). If not present the notebook uses sklearn's built-in dataset.
- `linear_regression_california_model.pkl` — trained model saved by the notebook.
- `training_results.txt` — MAE / RMSE / R² results from training.

How to run locally
1. Create and activate Python venv inside the project folder (optional but recommended):
   - Windows PowerShell:
     ```powershell
     python -m venv .venv
     .\.venv\Scripts\Activate.ps1
     ```
2. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```
3. Open `task1_ml_linear_regression.ipynb` in Jupyter or VS Code and run cells top→bottom.
